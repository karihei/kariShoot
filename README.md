![img](img/screenshot.png)

### For develop

Prerequisites:
```
* Node.js
* npm
* sqlite3
```

### Preparation to run!

How to get src.

```bash
$ git clone --recursive https://github.com/karihei/kariShoot
$ cd kariShoot
```

```bash
$ cd server
$ npm install sqlite3 socket.io
$ node server.js # running
```
